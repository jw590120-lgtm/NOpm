## 一、测试环境

| 项目 | 说明 |
|------|------|
| 前端地址 | http://localhost:5174 |
| 后端地址 | http://localhost:3001 |
| 浏览器 | Chrome / Edge（推荐最新版 Chromium 内核） |
| 屏幕分辨率 | 建议 1920×1080 及以上，缩放比例 100% |
| 预置数据 | 3 个种子产品（N系列、Nplus、N三代），覆盖 2018–2045 时间跨度 |
| 前端技术栈 | Vite + React |
| 后端技术栈 | Node.js + Express + TypeScript（tsx 运行时） |

> **启动说明**：前端在项目根目录执行 `npx vite --host 0.0.0.0`；后端进入 `server` 目录后执行 `npx tsx src/index.ts`。前后端需同时运行，端口 5174 对 3001。
